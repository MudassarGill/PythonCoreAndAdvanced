# 🐍 Python Core and Advanced Concepts Revision

Welcome to the **Python Core and Advanced Revision** repository! 🚀

This repository is a comprehensive collection of Python scripts and Jupyter Notebooks created for practicing, revising, and mastering various core and advanced Python programming concepts. It serves as a valuable resource for anyone looking to strengthen their Python skills and serves as a solid foundation for robust application development.

## 📚 Topics Covered

Here is a detailed breakdown of the key concepts revised in this repository, along with their respective source files:

### 1. Python Fundamentals
- **File:** `fundamental.ipynb`
- **Description:** Covers the basic building blocks of Python, including syntax, data types, control flow (loops and conditionals), operators, and functions.

### 2. Object-Oriented Programming (OOP)
- **File:** `OOp_python.ipynb`
- **Description:** A deep dive into Object-Oriented Programming concepts such as Classes, Objects, Inheritance, Polymorphism, Encapsulation, and Abstraction.

### 3. Exception Handling
- **File:** `TryExcpt.ipynb`
- **Description:** Implementation of robust error handling using `try`, `except`, `else`, and `finally` blocks to manage runtime errors gracefully and prevent application crashes.

### 4. File Handling
- **Files:** `file_handling.py`, `file.txt`
- **Description:** Practical examples of opening, reading, writing, and appending data to text files. Efficient resource management using context managers (`with` statements).

### 5. OS Module Subsystem
- **File:** `os.py`
- **Description:** Demonstrates how to interact with the underlying operating system, manage directories, list files, and perform system-level tasks using Python's built-in `os` module.

### 6. Modules and Packages
- **Files:** `modules.py`, `modules2.py`, `__init__.py`
- **Description:** Examples of creating reusable code components (modules) and importing them into other scripts to maintain a clean, DRY (Don't Repeat Yourself), and modular codebase.

### 7. Application Logging
- **Files:** `loggingProject.py`, `app.log`
- **Description:** Understanding how to use Python's built-in `logging` module to track application behavior, appropriately set varying log levels (INFO, DEBUG, ERROR), and write logs to persistent tracking files.

## 🛠️ Mini Projects Included

To practically apply the learned concepts, the following structured mini-projects are included:

- **Bank Management System (`project.py`)**: A console-based application demonstrating Object-Oriented Architecture (Abstract Classes, Inheritance). It is integrated with thorough logging to track operations across Current and Savings Accounts (deposits, withdrawals, and balance inquiries).
- **Interactive Calculator (`calculator.py`)**: A functional calculator application enhanced with logging capabilities to record mathematical operations and inputs.

## 🚀 How to Use / Run Locally

1. **Clone the repository:**
   ```bash
   git clone <'https://github.com/MudassarGill/PythonCoreAndAdvanced.git'>
   cd PythonCoreAndAdvanced
   ```

2. **Jupyter Notebooks:**
   Install Jupyter and open the `.ipynb` files to interactively run the core fundamental, OOP, and Exception Handling code blocks.
   ```bash
   pip install notebook
   jupyter notebook
   ```

3. **Python Scripts:**
   Run the `.py` scripts directly from your terminal using Python. For example:
   ```bash
   python project.py
   python file_handling.py
   ```
